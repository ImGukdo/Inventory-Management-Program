# 재고 관리 프로그램
관리자와 사용자에게 재고에 대한 정보를 제공한다.
## 소개
관리자는 재고 입출 및 프로그램을 관리할 수 있다.   
사용자는 재고 조회 및 출고를 요청할 수 있다.    
또한 게시판을 이용하여 관리자와 사용자 간의
커뮤니케이션이 가능하다.
